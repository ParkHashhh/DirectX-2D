1. 각 Manager의 Find 함수에서는 들어온 키로만 처리한다.
2. 각 Manager의 생성 함수에서는 들어온 키로만 처리한다.
3. WorldAssetManager 클래스에서 Find 함수의 생성 함수는
키를 만들어서 처리한다. 단, 생성 함수에서 Find함수를 호출하고
사용하는 경우 Map을 이용한 직접 탐색으로 변경한다.
4. Component들에서는 키를 만들지 않는다. 단, Animation2DComponent
는 Map 안에 직접 키를 이용하여 Animation2DSequence를 저장하고
있기 때문에 Sequence를 얻어올 필요가 있을 경우 Key를 만들어준다.
